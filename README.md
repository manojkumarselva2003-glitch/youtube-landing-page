# YouTube Landing Page Clone

A clone of the YouTube home page built using HTML and CSS.

## Features

- Responsive Navbar with search bar and icons
- Sidebar with navigation links and subscriptions
- Filter chips (horizontally scrollable)
- Video grid in 3 column x 3 row layout
- Video cards with thumbnails, channel profile pictures, title, views and upload time
- Responsive design — works on desktop, tablet and mobile

## Technologies Used

- HTML5
- CSS3
  - Flexbox (navbar, sidebar, video cards)
  - CSS Grid (video grid layout)
  - Responsive design using media queries

## How to Run

1. Clone or download the repository
2. Open `index.html` in any browser
3. No installation or setup needed
